# Sunny-Land-Florest
Repositório criado para estudo e consolidação da base de desenvolvimento na Unity 6, utilizando o projeto Sunny Land Florest, um jogo 2D em pixel art. O projeto aborda setup inicial, importação e manipulação de assets, criação de cenários, colisores, movimentação e interação do personagem, animações, cenas e lógica em C#.
